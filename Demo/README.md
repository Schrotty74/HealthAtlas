# HealthAtlas Demo

`AppleHealthDemo/Export.xml` is a fictional Apple Health export for trying the current HealthAtlas import and visualisation flow. It contains no personal health data.

Included sample types: steps, heart rate, body mass, active energy, walking/running distance and sleep-analysis records.

To try it, open HealthAtlas, choose **Import Apple Health…**, then select `Demo/AppleHealthDemo/Export.xml`. Use **Sources** to select types, **Overview** to choose 4, 8 or 12 cards, and **Trends** to select a type, a period and individual data points.
